## Help
### Call localhost from AVD app
Option A:
1. Go to Android\Sdk\platform-tools
2. Run command
   ./adb reverse tcp:7088 tcp:7088
   where 5204 is a port you wont to call.
Option B:
1. Use IP '10.0.2.2'
